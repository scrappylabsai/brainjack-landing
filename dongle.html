FAQ

Do I need the dongle if I install the service?

No. The free software service does the same thing over WiFi. The dongle is for machines where you can't (or don't want to) install anything.

Does it work with locked-down corporate machines?

Yes. That's the killer use case. The dongle appears as a standard USB keyboard — same as a Logitech or Dell. No software, no network traffic, nothing for IT to block.

What's inside?

ESP32-S3 microcontroller with native USB HID support, WiFi, and Bluetooth. Open hardware — you can build your own from the specs.

Do I need an iPhone?

Currently yes — the BrainJack app is iOS only (TestFlight beta). Android is planned.
